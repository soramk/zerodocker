🖥️

Frontend / Backend

Python (Flask) または Node.js (Express)

  • テキスト入力と投稿機能
  • 一覧表示機能
💾

Database

PostgreSQL または Redis

  • データの永続化
  • ブラウザを閉じても消えない
🐳

Infrastructure

Docker & Docker Compose

  • アプリとDBの連携
  • 一発起動の仕組み
